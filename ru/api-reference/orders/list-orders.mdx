## Список заказов

Возвращает постраничный список ваших заказов, начиная с новых.

### Параметры запроса

<ParamField query="page" type="number">
  Номер страницы (по умолчанию 1)
</ParamField>

<ParamField query="page_size" type="number">
  Результатов на страницу (по умолчанию 50, максимум 100)
</ParamField>

<ParamField query="status" type="string">
  Фильтр по статусу (например, `filled`, `pending`, `failed`)
</ParamField>

<ParamField query="type" type="string">
  Фильтр по типу заказа — `energy` или `bandwidth`
</ParamField>

### Ответ

Возвращает `orders` (массив объектов заказа), `total`, `page` и `page_size`.

### Пример

```bash
curl "https://api.tronrental.com/v1/orders?type=energy&status=filled&page=1" \
  -H "X-API-Key: your_api_key"
```
