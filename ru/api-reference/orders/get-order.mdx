## Получить заказ

Возвращает текущий статус и полные детали заказа энергии или bandwidth.

### Параметры пути

<ParamField path="order_id" type="number" required>
  ID заказа — значение `id`, которое возвращают Buy Energy или Buy Bandwidth.
</ParamField>

### Ответ

Возвращает полный объект заказа, включая `status`, `type`, `volume`, `price_trx`, `delegate_txid` и временные метки.

### Статусы заказа

| Статус | Описание |
|--------|-------------|
| `pending` | Заказ принят, делегирование в процессе |
| `filled` | Энергия или bandwidth успешно делегированы |
| `failed` | Делегирование не удалось |
| `reclaimed` | Bandwidth возвращён после окончания срока аренды (только заказы bandwidth) |

### Пример

```bash
curl https://api.tronrental.com/v1/orders/12345 \
  -H "X-API-Key: your_api_key"
```
