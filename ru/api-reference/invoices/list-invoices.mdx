## Список инвойсов

Возвращает пагинированный список ваших инвойсов.

### Параметры запроса

<ParamField query="page" type="number" default="1">
  Номер страницы
</ParamField>

<ParamField query="status" type="string">
  Фильтр по статусу: `pending`, `paid`, `delegated`, `expired`, `failed`
</ParamField>

### Ответ

Возвращает пагинированный объект с массивом `invoices` и общим количеством `total`.

### Пример

```bash
curl "https://api.tronrental.com/v1/invoices/my?page=1&status=delegated" \
  -H "X-API-Key: your_api_key"
```
