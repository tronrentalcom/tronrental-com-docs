## Получить инвойс

Возвращает детали и текущий статус инвойса.

### Параметры пути

<ParamField path="invoice_id" type="number" required>
  ID инвойса
</ParamField>

### Ответ

Возвращает полный объект инвойса, включая `status`, `payment_address`, `price_trx`, `price_usdt`, `tx_hash` и временные метки.

### Статусы инвойсов

| Статус | Описание |
|--------|----------|
| `pending` | Ожидание оплаты |
| `paid` | Оплата получена, делегирование в процессе |
| `delegated` | Энергия успешно делегирована |
| `expired` | Оплата не получена до истечения срока |
| `failed` | Делегирование не удалось (оплата возвращена) |

### Пример

```bash
curl https://api.tronrental.com/v1/invoices/12345 \
  -H "X-API-Key: your_api_key"
```
