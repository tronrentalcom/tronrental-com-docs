## Список аренд с автопродлением

Возвращает ваши аренды с автопродлением, начиная с новых.

### Параметры запроса

<ParamField query="status" type="string">
  Фильтр по статусу: `active`, `cancelled` или `ended_insufficient_balance`
</ParamField>

### Ответ

Возвращает `rentals` — массив объектов аренды с `status`, `volume`, `daily_trx`, `total_charged_trx`, `next_billing_at` и временными метками.

### Статусы аренды

| Статус | Описание |
|--------|-------------|
| `active` | Делегировано, списание раз в сутки |
| `cancelled` | Отменено пользователем (или админом) |
| `ended_insufficient_balance` | Завершено автоматически — баланса не хватило на следующий день |

### Пример

```bash
curl "https://api.tronrental.com/v1/bandwidth/rentals?status=active" \
  -H "X-API-Key: your_api_key"
```
