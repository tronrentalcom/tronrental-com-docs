## Smart Mode

Smart Mode автоматически предоставляет энергию для каждого исходящего перевода USDT с отслеживаемого адреса. Ручные вызовы API не нужны — просто активируйте и переводите.

### Как это работает

1. **Активируйте** Smart Mode для адреса TRON
2. Каждый раз, когда этот адрес отправляет USDT, энергия **автоматически делегируется**
3. Ежедневная подписка + комиссия за каждый перевод списываются с вашего баланса
4. Если баланс закончится, подписка приостанавливается (не удаляется)

### Ценообразование

| Комиссия | Сумма |
|----------|-------|
| Ежедневная подписка | 3.0 TRX/день |
| За перевод USDT | 2.7 TRX × множитель |
| BW Guarantee (опционально) | +0.3 TRX/перевод |

Множитель за перевод равен `1` для адресов, которые уже содержат USDT (65K энергии), и `2` для первого получения USDT (131K энергии).

<Tip>
  **BW Guarantee** (включена по умолчанию) делегирует 350 bandwidth для покрытия стоимости bandwidth транзакции.
  Без неё отправителю потребуется ~0.35 TRX в виде bandwidth или застейканного TRX.
</Tip>

<Info>
  Цены выше являются значениями по умолчанию. Используйте [`GET /smart-mode/fees`](/api-reference/smart-mode/get-fees) для проверки ваших актуальных тарифов — возможно индивидуальное ценообразование.
</Info>

### Лимиты

- Максимум 5 активных подписок на аккаунт
- Индивидуальное ценообразование доступно для пользователей с большим объёмом (обратитесь в поддержку)
