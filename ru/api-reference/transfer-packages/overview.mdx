## Пакеты переводов

Пакеты переводов — это предоплаченные пакеты переводов USDT. Купите пакет на N переводов, и каждый исходящий перевод USDT с адреса автоматически получает энергию, пока пакет не будет израсходован.

### Как это работает

1. **Купите пакет** для определённого адреса TRON (например, 500 переводов)
2. Каждый исходящий перевод USDT **автоматически получает энергию**
3. Счётчик пакета уменьшается с каждым переводом
4. Когда пакет исчерпан, мониторинг прекращается

### Ценообразование

| Размер пакета | Цена за перевод |
|---------------|-----------------|
| 1 – 10 000 переводов | 2.7 TRX |
| BW Guarantee (опционально) | +0.3 TRX/перевод |

**Готовые варианты:** 500, 1 000, 5 000, 10 000

### Пакеты переводов vs Smart Mode

| Параметр | Пакеты переводов | Smart Mode |
|----------|------------------|------------|
| Оплата | Предоплата (разово) | Ежедневная подписка + за перевод |
| Срок действия | До исчерпания переводов | Бессрочно |
| Подходит для | Предсказуемый объём | Постоянное использование |

<Warning>
  Адрес не может одновременно иметь активный пакет переводов и Smart Mode.
</Warning>
