## Активация адреса

Активирует адрес TRON, который ещё не получал транзакций. Требуется перед тем, как адрес сможет получать токены.

**Стоимость: 1.1 TRX** (списывается с вашего баланса)

### Тело запроса

<ParamField body="address" type="string" required>
  Адрес TRON для активации (формат T...)
</ParamField>

### Ответ

<ResponseField name="success" type="boolean">
  Успешна ли активация
</ResponseField>

<ResponseField name="tx_hash" type="string">
  Хеш транзакции активации
</ResponseField>

### Пример

```bash
curl -X POST https://api.tronrental.com/v1/energy/activate \
  -H "X-API-Key: your_api_key" \
  -H "Content-Type: application/json" \
  -d '{"address": "TNewInactiveAddress..."}'
```

```json Response
{
  "success": true,
  "tx_hash": "a1b2c3d4e5f6..."
}
```
