## Получить цены

Возвращает текущие рыночные цены на энергию и bandwidth.

**Этот эндпоинт публичный — аутентификация не требуется.**

### Ответ

<ResponseField name="energy_price_sun" type="number">
  Текущая цена энергии в SUN за единицу
</ResponseField>

<ResponseField name="energy_price_1k_trx" type="string">
  Цена за 1 000 единиц энергии в TRX
</ResponseField>

<ResponseField name="bw_price_1k_trx" type="string">
  Цена за 1 000 единиц bandwidth в TRX
</ResponseField>

<ResponseField name="trx_usd_rate" type="string">
  Текущий курс TRX/USD
</ResponseField>

<ResponseField name="burn_sun_per_energy" type="number">
  Сетевая стоимость сжигания — SUN за единицу энергии (в настоящее время 100)
</ResponseField>

<ResponseField name="min_energy" type="number">
  Минимальное количество энергии за заказ
</ResponseField>

<ResponseField name="max_energy" type="number">
  Максимальное количество энергии за заказ
</ResponseField>

### Пример

<CodeGroup>
```bash cURL
curl https://api.tronrental.com/v1/prices
```

```python Python
import requests
resp = requests.get("https://api.tronrental.com/v1/prices")
print(resp.json())
```

```javascript JavaScript
const resp = await fetch("https://api.tronrental.com/v1/prices");
const data = await resp.json();
console.log(data);
```
</CodeGroup>
