## Получить цены

Возвращает текущие рыночные цены на энергию и bandwidth.

**Этот эндпоинт публичный — аутентификация не требуется.**

### Ответ

<ResponseField name="energy_trx" type="object">
  Цена энергии в TRX за 65 000 единиц, с ключом по длительности (`1h`)
</ResponseField>

<ResponseField name="energy_sun" type="object">
  Цена энергии в SUN за 1 единицу, с ключом по длительности (`1h`)
</ResponseField>

<ResponseField name="energy_usd" type="object">
  Цена энергии в USD за 65 000 единиц, с ключом по длительности (`1h`)
</ResponseField>

<ResponseField name="energy_volume" type="number">
  Объём, используемый для расчёта цены энергии (65 000)
</ResponseField>

<ResponseField name="bandwidth_trx" type="object">
  Цена bandwidth в TRX за 350 единиц, с ключом по длительности (`1h`)
</ResponseField>

<ResponseField name="bandwidth_sun" type="object">
  Цена bandwidth в SUN за 1 единицу, с ключом по длительности (`1h`)
</ResponseField>

<ResponseField name="bandwidth_usd" type="object">
  Цена bandwidth в USD за 350 единиц, с ключом по длительности (`1h`)
</ResponseField>

<ResponseField name="bandwidth_volume" type="number">
  Объём, используемый для расчёта цены bandwidth (350)
</ResponseField>

<ResponseField name="trx_usd_rate" type="string">
  Текущий курс TRX/USD
</ResponseField>

<ResponseField name="burn_cost_trx" type="string">
  Стоимость сжигания TRX за 65 000 энергии (без аренды)
</ResponseField>

<ResponseField name="burn_cost_usd" type="string">
  Та же стоимость сжигания в USD
</ResponseField>

<ResponseField name="savings_percent" type="string">
  Процент экономии при аренде по сравнению со сжиганием
</ResponseField>

### Пример

<CodeGroup>
```bash cURL
curl https://api.tronrental.com/v1/prices
```

```python Python
import requests
resp = requests.get("https://api.tronrental.com/v1/prices")
print(resp.json())
```

```javascript JavaScript
const resp = await fetch("https://api.tronrental.com/v1/prices");
const data = await resp.json();
console.log(data);
```
</CodeGroup>
