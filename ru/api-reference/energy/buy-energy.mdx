## Купить энергию

Делегирует энергию на указанный адрес TRON. Стоимость списывается с баланса вашего аккаунта.

### Тело запроса

<ParamField body="address" type="string" required>
  Целевой адрес TRON (формат T..., 34 символа)
</ParamField>

<ParamField body="energy_amount" type="number" required>
  Количество делегируемой энергии (мин. 32 000, макс. 5 000 000)
</ParamField>

<ParamField body="duration_hours" type="number" required>
  Длительность аренды в часах (1 или 24)
</ParamField>

### Ответ

<ResponseField name="order_id" type="number">
  Уникальный идентификатор заказа
</ResponseField>

<ResponseField name="address" type="string">
  Целевой адрес
</ResponseField>

<ResponseField name="energy_amount" type="number">
  Делегированная энергия
</ResponseField>

<ResponseField name="duration_hours" type="number">
  Длительность аренды
</ResponseField>

<ResponseField name="price_trx" type="string">
  Общая стоимость в TRX
</ResponseField>

<ResponseField name="status" type="string">
  Статус заказа: `pending`, `filled`, `failed`
</ResponseField>

### Пример

<CodeGroup>
```bash cURL
curl -X POST https://api.tronrental.com/v1/energy/buy \
  -H "X-API-Key: your_api_key" \
  -H "Content-Type: application/json" \
  -d '{
    "address": "TYourRecipientAddress1234567890abc",
    "energy_amount": 65000,
    "duration_hours": 1
  }'
```

```python Python
import requests

resp = requests.post(
    "https://api.tronrental.com/v1/energy/buy",
    headers={"X-API-Key": "your_api_key"},
    json={
        "address": "TYourRecipientAddress1234567890abc",
        "energy_amount": 65000,
        "duration_hours": 1,
    },
)
print(resp.json())
```

```javascript JavaScript
const resp = await fetch("https://api.tronrental.com/v1/energy/buy", {
  method: "POST",
  headers: {
    "X-API-Key": "your_api_key",
    "Content-Type": "application/json",
  },
  body: JSON.stringify({
    address: "TYourRecipientAddress1234567890abc",
    energy_amount: 65000,
    duration_hours: 1,
  }),
});
console.log(await resp.json());
```
</CodeGroup>

```json Response
{
  "order_id": 12345,
  "address": "TYourRecipientAddress1234567890abc",
  "energy_amount": 65000,
  "duration_hours": 1,
  "price_trx": "2.75",

```

<Tip>
  **65 000 энергии** покрывает один перевод USDT на адрес, который уже содержит USDT.
  Используйте **131 000** для первого получения USDT (создание нового слота хранения).
</Tip>
