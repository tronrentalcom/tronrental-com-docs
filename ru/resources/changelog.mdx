## Список изменений

### Март 2026

- **Пакеты переводов** — предоплаченные пакеты переводов USDT
- **BW Guarantee** — опциональное делегирование bandwidth для Smart Mode и пакетов переводов
- **Вебхуки** — уведомления в реальном времени о событиях аккаунта
- **API вывода средств** — вывод TRX на любой адрес TRON

### Февраль 2026

- **Smart Mode** — автоматическая энергия для переводов USDT
- **API инвойсов** — создание платёжных инвойсов на энергию
- **API-ключи** — поддержка белого списка IP
- **API bandwidth** — покупка bandwidth для адресов TRON

### Январь 2026 — Первый релиз

- API покупки энергии
- Управление аккаунтом (баланс, депозиты, профиль)
- Цены в реальном времени
