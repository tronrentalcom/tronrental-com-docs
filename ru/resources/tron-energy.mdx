## Что такое энергия TRON?

Энергия — это ресурс в блокчейне TRON, необходимый для выполнения смарт-контрактов. Каждый перевод TRC-20 токенов (например, USDT) потребляет энергию.

### Расход энергии для переводов USDT

| Сценарий | Необходимая энергия |
|----------|---------------------|
| Получатель **имеет** USDT | **~65 000** (точно: 64 285) |
| Получатель **не имеет** USDT | **~131 000** (точно: 130 285) |

Разница обусловлена созданием слота хранения — отправка USDT новому держателю требует создания новой записи в контракте токена.

### Без энергии: сжигается TRX

Если у вас нет энергии, сеть сжигает TRX с вашего аккаунта для покрытия расходов:

```
64,285 energy × 100 SUN/energy = 6,428,500 SUN = 6.43 TRX
```

По текущим курсам это примерно **$1.50 за один перевод USDT**.

### С арендованной энергией: экономия 60-80%

Аренда энергии стоит примерно **2.5-3.0 TRX за перевод** — экономия 50-60% по сравнению со сжиганием.

### Как работает аренда

Энергия предоставляется через **делегирование ресурсов** — функцию сети TRON, при которой один аккаунт временно передаёт свои застейканные ресурсы другому. Энергия появляется на целевом адресе и расходуется при следующей транзакции.
