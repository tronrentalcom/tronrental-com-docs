## Лимиты запросов

Эндпоинты API имеют ограничения по частоте запросов для обеспечения справедливого использования и стабильности сервиса.

### Лимиты по эндпоинтам

| Эндпоинт | Лимит |
|----------|-------|
| `GET /prices` | 60 запросов/минута |
| `POST /energy/buy` | 30 запросов/минута |
| `POST /bandwidth/buy` | 30 запросов/минута |
| `POST /invoices` | 20 запросов/минута |
| `POST /smart-mode/activate` | 10 запросов/минута |
| `POST /account/withdrawal` | 1 запрос/минута |
| Остальные эндпоинты | 60 запросов/минута |

### Заголовки лимитов

Ответы содержат информацию о лимитах в заголовках:

```
X-RateLimit-Limit: 60
X-RateLimit-Remaining: 58
X-RateLimit-Reset: 1709568000
```

### Превышение лимитов

При превышении лимита вы получите HTTP `429`:

```json
{
  "detail": "Rate limit exceeded. Try again in 30 seconds."
}
```

Используйте экспоненциальную задержку при повторных попытках. Не повторяйте запрос немедленно.
