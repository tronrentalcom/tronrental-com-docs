### Базовый URL

Все API-запросы используют следующий базовый URL:

```
https://api.tronrental.com/v1
```

### Быстрый пример

```bash
curl -X POST https://api.tronrental.com/v1/energy/buy \
  -H "X-API-Key: your_api_key" \
  -H "Content-Type: application/json" \
  -d '{
    "address": "TYourTronAddress...",
    "energy_amount": 65000,
    "duration_hours": 1
  }'
```

```json Response
{
  "order_id": 12345,
  "address": "TYourTronAddress...",
  "energy_amount": 65000,
  "duration_hours": 1,
  "price_trx": "2.75",
  "status": "pending"
}
```

## Следующие шаги

<Steps>
  <Step title="Получите API-ключ">
    Зарегистрируйтесь на [tronrental.com](https://tronrental.com/auth/register) и создайте API-ключ в [личном кабинете](https://tronrental.com/dashboard/api).

  <Step title="Проверьте текущие цены">

  <Step title="Сделайте первую покупку">
    Используйте `POST /api/v1/energy/buy` для покупки энергии на адрес TRON.
  </Step>
</Steps>
