## Listar órdenes

Devuelve una lista paginada de tus órdenes, las más recientes primero.

### Parámetros de consulta

<ParamField query="page" type="number">
  Número de página (predeterminado 1)
</ParamField>

<ParamField query="page_size" type="number">
  Resultados por página (predeterminado 50, máximo 100)
</ParamField>

<ParamField query="status" type="string">
  Filtrar por estado (p. ej. `filled`, `pending`, `failed`)
</ParamField>

<ParamField query="type" type="string">
  Filtrar por tipo de orden — `energy` o `bandwidth`
</ParamField>

### Respuesta

Devuelve `orders` (array de objetos de orden), `total`, `page` y `page_size`.

### Ejemplo

```bash
curl "https://api.tronrental.com/v1/orders?type=energy&status=filled&page=1" \
  -H "X-API-Key: your_api_key"
```
