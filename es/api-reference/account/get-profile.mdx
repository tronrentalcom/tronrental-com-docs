## Obtener perfil

Devuelve la informacion del perfil de tu cuenta.

### Respuesta

<ResponseField name="id" type="number">
  ID de usuario
</ResponseField>

<ResponseField name="email" type="string">
  Direccion de correo electronico (si esta configurada)
</ResponseField>

<ResponseField name="tron_address" type="string">
  Direccion TRON vinculada (si esta configurada)
</ResponseField>

<ResponseField name="balance_trx" type="string">
  Saldo de la cuenta en TRX
</ResponseField>

<ResponseField name="totp_enabled" type="boolean">
  Si la autenticacion de dos factores esta habilitada
</ResponseField>

<ResponseField name="created_at" type="string">
  Marca de tiempo de creacion de la cuenta
</ResponseField>

### Ejemplo

```bash
curl https://api.tronrental.com/v1/account/profile \
  -H "X-API-Key: your_api_key"
```

```json Response
{
  "id": 1,
  "email": "user@example.com",
  "tron_address": "TYourAddress...",
  "balance_trx": "150.50",
  "totp_enabled": false,
  "created_at": "2026-01-15T10:00:00Z"
}
```
