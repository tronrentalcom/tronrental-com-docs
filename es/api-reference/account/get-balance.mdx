## Obtener saldo

Devuelve tu saldo actual en TRX.

### Respuesta

<ResponseField name="balance_trx" type="string">
  Saldo de la cuenta en TRX (cadena decimal)
</ResponseField>

### Ejemplo

```bash
curl https://api.tronrental.com/v1/account/balance \
  -H "X-API-Key: your_api_key"
```

```json
{
  "balance_trx": "156.50"
}
```
