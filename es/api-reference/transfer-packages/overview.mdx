## Paquetes de Transferencias

Los Paquetes de Transferencias son paquetes prepagados de transferencias USDT. Compra un paquete de N transferencias, y cada transferencia USDT saliente desde la direccion recibe energia automaticamente hasta que el paquete se agote.

### Como funciona

1. **Compra un paquete** para una direccion TRON especifica (ej., 500 transferencias)
2. Cada transferencia USDT saliente **recibe energia automaticamente**
3. El contador del paquete disminuye con cada transferencia
4. Cuando el paquete se agota, el monitoreo se detiene

### Precios

| Tamano del paquete | Precio por transferencia |
|--------------------|--------------------------|
| 1 – 10,000 transferencias | 2.7 TRX |
| Garantia de BW (opcional) | +0.3 TRX/transferencia |

**Preselecciones rapidas:** 500, 1,000, 5,000, 10,000

### Paquetes de Transferencias vs Smart Mode

| Caracteristica | Paquetes de Transferencias | Smart Mode |
|----------------|---------------------------|------------|
| Facturacion | Prepago (unico) | Suscripcion diaria + por transferencia |
| Duracion | Hasta agotar transferencias | Continuo |
| Ideal para | Volumen predecible | Uso continuo |

<Warning>
  Una direccion no puede tener un Paquete de Transferencias activo y Smart Mode al mismo tiempo.
</Warning>
