## Obtener niveles de paquetes

Devuelve los tamanos de paquetes preestablecidos, precios y limites disponibles.

### Respuesta

<ResponseField name="tiers" type="array">
  Array de opciones de paquetes preestablecidos con tamano y precio total
</ResponseField>

<ResponseField name="single_transfer_price" type="string">
  Precio por una transferencia individual en TRX
</ResponseField>

<ResponseField name="bw_fee_per_transfer" type="string">
  Tarifa de Garantia de BW por transferencia en TRX
</ResponseField>

<ResponseField name="min_custom_size" type="number">
  Tamano minimo de paquete personalizado (1)
</ResponseField>

<ResponseField name="max_custom_size" type="number">
  Tamano maximo de paquete personalizado (10,000)
</ResponseField>

### Ejemplo

```bash
curl https://api.tronrental.com/v1/transfer-packages/tiers \
  -H "X-API-Key: your_api_key"
```

```json Response
{
  "tiers": [
    { "size": 100, "price_trx": "270.00" },
    { "size": 500, "price_trx": "1350.00" },
    { "size": 1000, "price_trx": "2700.00" }
  ],
  "single_transfer_price": "2.70",
  "bw_fee_per_transfer": "0.30",
  "min_custom_size": 1,
  "max_custom_size": 10000
}
```
