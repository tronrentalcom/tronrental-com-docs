## Alternar suscripcion

Pausa una suscripcion activa o reanuda una pausada. Mientras esta pausada, no se cobran tarifas y las transferencias no se monitorean.

### Parametros de ruta

<ParamField path="subscription_id" type="number" required>
  ID de la suscripcion
</ParamField>

### Respuesta

Devuelve el objeto de suscripcion actualizado con el nuevo `status`.

### Ejemplo

```bash
curl -X POST https://api.tronrental.com/v1/smart-mode/42/toggle \
  -H "X-API-Key: your_api_key"
```

```json Response
{
  "id": 42,
  "address": "TYourAddress...",
  "label": "Main wallet",
  "status": "paused",
  "bw_guarantee": true,
  "created_at": "2026-03-05T12:00:00Z"
}
```
