## Obtener precios

Devuelve los precios de mercado actuales para energia y ancho de banda.

**Este endpoint es publico — no requiere autenticacion.**

### Respuesta

<ResponseField name="energy_trx" type="object">
  Precio de energia en TRX para 65,000 unidades, por duracion (`1h`)
</ResponseField>

<ResponseField name="energy_sun" type="object">
  Precio de energia en SUN por 1 unidad, por duracion (`1h`)
</ResponseField>

<ResponseField name="energy_usd" type="object">
  Precio de energia en USD para 65,000 unidades, por duracion (`1h`)
</ResponseField>

<ResponseField name="energy_volume" type="number">
  Volumen utilizado para el calculo del precio de energia (65,000)
</ResponseField>

<ResponseField name="bandwidth_trx" type="object">
  Precio de ancho de banda en TRX para 350 unidades, por duracion (`1h`)
</ResponseField>

<ResponseField name="bandwidth_sun" type="object">
  Precio de ancho de banda en SUN por 1 unidad, por duracion (`1h`)
</ResponseField>

<ResponseField name="bandwidth_usd" type="object">
  Precio de ancho de banda en USD para 350 unidades, por duracion (`1h`)
</ResponseField>

<ResponseField name="bandwidth_volume" type="number">
  Volumen utilizado para el calculo del precio de ancho de banda (350)
</ResponseField>

<ResponseField name="trx_usd_rate" type="string">
  Tasa de cambio actual TRX/USD
</ResponseField>

<ResponseField name="burn_cost_trx" type="string">
  Costo de quemar TRX para 65,000 de energia (sin alquiler)
</ResponseField>

<ResponseField name="burn_cost_usd" type="string">
  Mismo costo de quema en USD
</ResponseField>

<ResponseField name="savings_percent" type="string">
  Porcentaje de ahorro al alquilar vs quemar
</ResponseField>

### Ejemplo

<CodeGroup>
```bash cURL
curl https://api.tronrental.com/v1/prices
```

```python Python
import requests
resp = requests.get("https://api.tronrental.com/v1/prices")
print(resp.json())
```

```javascript JavaScript
const resp = await fetch("https://api.tronrental.com/v1/prices");
const data = await resp.json();
console.log(data);
```
</CodeGroup>
