## Obtener factura

Devuelve los detalles y el estado actual de una factura.

### Parametros de ruta

<ParamField path="invoice_id" type="number" required>
  ID de la factura
</ParamField>

### Respuesta

Devuelve el objeto completo de la factura incluyendo `status`, `payment_address`, `price_trx`, `price_usdt`, `tx_hash` y marcas de tiempo.

### Estados de la factura

| Estado | Descripcion |
|--------|-------------|
| `pending` | Esperando pago |
| `paid` | Pago recibido, delegacion en progreso |
| `delegated` | Energia delegada exitosamente |
| `expired` | No se recibio pago antes de la expiracion |
| `failed` | La delegacion fallo (pago reembolsado) |

### Ejemplo

```bash
curl https://api.tronrental.com/v1/invoices/12345 \
  -H "X-API-Key: your_api_key"
```
