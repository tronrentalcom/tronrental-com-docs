## Registro de cambios

### Marzo 2026

- **Paquetes de Transferencias** — paquetes prepagados de transferencias USDT
- **Garantia de BW** — delegacion opcional de ancho de banda para Smart Mode y Paquetes de Transferencias
- **Webhooks** — notificaciones en tiempo real para eventos de la cuenta
- **API de Retiros** — retira TRX a cualquier direccion TRON

### Febrero 2026

- **Smart Mode** — energia automatica para transferencias USDT
- **API de Facturas** — crea facturas de pago para energia
- **Claves API** — soporte de lista blanca de IP
- **API de Ancho de Banda** — compra ancho de banda para direcciones TRON

### Enero 2026 — Lanzamiento inicial

- API de compra de energia
- Gestion de cuenta (saldo, depositos, perfil)
- Precios en tiempo real
