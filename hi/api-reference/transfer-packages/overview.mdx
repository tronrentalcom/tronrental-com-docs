## Transfer Packages

Transfer Packages USDT ट्रांसफर के प्रीपेड बंडल हैं। N ट्रांसफर का पैकेज खरीदें, और एड्रेस से प्रत्येक आउटगोइंग USDT ट्रांसफर को पैकेज समाप्त होने तक स्वचालित रूप से एनर्जी प्राप्त होती है।

### यह कैसे काम करता है

1. एक विशिष्ट TRON एड्रेस के लिए **पैकेज खरीदें** (उदा., 500 ट्रांसफर)
2. प्रत्येक आउटगोइंग USDT ट्रांसफर को **स्वचालित रूप से एनर्जी मिलती है**
3. प्रत्येक ट्रांसफर के साथ पैकेज काउंटर घटता है
4. जब पैकेज समाप्त हो जाता है, मॉनिटरिंग रुक जाती है

### मूल्य निर्धारण

| पैकेज आकार | प्रति ट्रांसफर मूल्य |
|-------------|-------------------|
| 1 – 10,000 ट्रांसफर | 2.7 TRX |
| BW Guarantee (वैकल्पिक) | +0.3 TRX/ट्रांसफर |

**क्विक-सिलेक्ट प्रीसेट:** 500, 1,000, 5,000, 10,000

### Transfer Packages बनाम Smart Mode

| सुविधा | Transfer Packages | Smart Mode |
|---------|------------------|------------|
| बिलिंग | प्रीपेड (एकमुश्त) | दैनिक सब्सक्रिप्शन + प्रति-ट्रांसफर |
| अवधि | ट्रांसफर उपयोग होने तक | चालू |
| सबसे उपयुक्त | अनुमानित वॉल्यूम | निरंतर उपयोग |

<Warning>
  एक एड्रेस पर एक ही समय में सक्रिय Transfer Package और Smart Mode दोनों नहीं हो सकते।
</Warning>
