## Smart Mode

Smart Mode एक मॉनिटर किए गए एड्रेस से हर आउटगोइंग USDT ट्रांसफर के लिए स्वचालित रूप से एनर्जी प्रदान करता है। कोई मैनुअल API कॉल की आवश्यकता नहीं — बस सक्रिय करें और ट्रांसफर करें।

### यह कैसे काम करता है

1. TRON एड्रेस के लिए Smart Mode **सक्रिय** करें
2. हर बार जब वह एड्रेस USDT भेजता है, एनर्जी **स्वचालित रूप से डेलिगेट** की जाती है
3. दैनिक सब्सक्रिप्शन शुल्क + प्रति-ट्रांसफर शुल्क आपके बैलेंस से काटा जाता है
4. यदि बैलेंस समाप्त हो जाता है, तो सब्सक्रिप्शन रोक दिया जाता है (हटाया नहीं जाता)

### मूल्य निर्धारण

| शुल्क | राशि |
|-----|--------|
| दैनिक सब्सक्रिप्शन | 3.0 TRX/दिन |
| प्रति USDT ट्रांसफर | 2.7 TRX × मल्टीप्लायर |
| BW Guarantee (वैकल्पिक) | +0.3 TRX/ट्रांसफर |

ट्रांसफर मल्टीप्लायर उन एड्रेस के लिए `1` है जिनके पास पहले से USDT है (65K एनर्जी) और पहली बार प्राप्त करने वालों के लिए `2` है (131K एनर्जी)।

<Tip>
  **BW Guarantee** (डिफ़ॉल्ट रूप से सक्षम) ट्रांज़ैक्शन की बैंडविड्थ लागत को कवर करने के लिए 350 बैंडविड्थ डेलिगेट करता है।
  इसके बिना, भेजने वाले को ~0.35 TRX मूल्य की बैंडविड्थ या फ्रोज़न TRX की आवश्यकता होती है।
</Tip>

<Info>
  ऊपर दिए गए मूल्य डिफ़ॉल्ट हैं। अपनी वास्तविक दरें जांचने के लिए [`GET /smart-mode/fees`](/api-reference/smart-mode/get-fees) का उपयोग करें — कस्टम प्राइसिंग लागू हो सकती है।
</Info>
