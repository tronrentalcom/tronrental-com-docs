## Webhook लॉग

डिबगिंग के लिए हालिया webhook डिलीवरी प्रयास लौटाता है।

### रिस्पॉन्स

लॉग एंट्री का ऐरे:

<ResponseField name="id" type="number">
  लॉग एंट्री ID
</ResponseField>

<ResponseField name="event_type" type="string">
  Webhook को ट्रिगर करने वाला इवेंट
</ResponseField>

<ResponseField name="payload" type="object">
  भेजा गया पेलोड
</ResponseField>

<ResponseField name="response_status" type="number">
  आपके सर्वर से HTTP स्टेटस कोड
</ResponseField>

<ResponseField name="attempts" type="number">
  डिलीवरी प्रयासों की संख्या
</ResponseField>

<ResponseField name="created_at" type="string">
  टाइमस्टैम्प
</ResponseField>

### उदाहरण

```bash
curl https://api.tronrental.com/v1/webhooks/logs \
  -H "X-API-Key: your_api_key"
```
