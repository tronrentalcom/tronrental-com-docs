## रेट लिमिट

उचित उपयोग और सेवा स्थिरता सुनिश्चित करने के लिए API एंडपॉइंट रेट-लिमिटेड हैं।

### एंडपॉइंट के अनुसार लिमिट

| एंडपॉइंट | लिमिट |
|----------|-------|
| `GET /prices` | 60 अनुरोध/मिनट |
| `POST /energy/buy` | 30 अनुरोध/मिनट |
| `POST /bandwidth/buy` | 30 अनुरोध/मिनट |
| `POST /invoices` | 20 अनुरोध/मिनट |
| `POST /smart-mode/activate` | 10 अनुरोध/मिनट |
| `POST /account/withdrawal` | 1 अनुरोध/मिनट |
| अन्य एंडपॉइंट | 60 अनुरोध/मिनट |

### रेट लिमिट हेडर

रिस्पॉन्स में हेडर में रेट लिमिट जानकारी शामिल होती है:

```
X-RateLimit-Limit: 60
X-RateLimit-Remaining: 58
X-RateLimit-Reset: 1709568000
```

### लिमिट पार होने पर

रेट लिमिट होने पर, आपको HTTP `429` प्राप्त होगा:

```json
{
  "detail": "Rate limit exceeded. Try again in 30 seconds."
}
```

पुनः प्रयास के लिए एक्सपोनेंशियल बैकऑफ़ लागू करें। तुरंत पुनः प्रयास न करें।
