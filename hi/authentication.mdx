## API कुंजी प्रमाणीकरण

सभी API अनुरोधों के लिए `X-API-Key` हेडर में API कुंजी पास करना आवश्यक है।

```bash
curl https://api.tronrental.com/v1/account/balance \
  -H "X-API-Key: your_api_key"
```

### API कुंजी प्राप्त करना

1. [tronrental.com](https://tronrental.com/auth/register) पर साइन अप करें
2. [डैशबोर्ड → API Keys](https://tronrental.com/dashboard/api) पर जाएं
3. **API कुंजी बनाएं** पर क्लिक करें
4. अपनी कुंजी कॉपी करें और सुरक्षित रूप से स्टोर करें — यह केवल एक बार दिखाई जाती है

### IP व्हाइटलिस्ट (वैकल्पिक)

अतिरिक्त सुरक्षा के लिए आप अपनी API कुंजी को विशिष्ट IP एड्रेस तक सीमित कर सकते हैं। कुंजी बनाते या संपादित करते समय डैशबोर्ड में यह कॉन्फ़िगर करें।

### कुंजी अनुमतियां

प्रत्येक API कुंजी के पास आपके अकाउंट के API ऑपरेशनों तक पूर्ण पहुंच होती है:
- एनर्जी और बैंडविड्थ खरीदें
- Smart Mode सब्सक्रिप्शन प्रबंधित करें
- इनवॉइस बनाएं
- बैलेंस और ट्रांज़ैक्शन इतिहास देखें

<Warning>
  अपनी API कुंजी को गोपनीय रखें। इसे क्लाइंट-साइड कोड, सार्वजनिक रिपॉजिटरी या URL में एक्सपोज़ न करें।
</Warning>

## रेट लिमिट

API कुंजियां प्रति एंडपॉइंट रेट-लिमिटेड हैं। विवरण के लिए [रेट लिमिट](/resources/rate-limits) देखें।

## त्रुटियां

प्रमाणीकरण त्रुटियां HTTP `401` लौटाती हैं:

```json
{
  "detail": "Invalid or missing API key"
}
```

यदि आपकी कुंजी अक्षम या ब्लॉक है, तो आपको HTTP `403` प्राप्त होगा।
