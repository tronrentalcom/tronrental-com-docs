## TronRental API में आपका स्वागत है

### बेस URL

सभी API अनुरोध निम्नलिखित बेस URL का उपयोग करते हैं:

```
https://api.tronrental.com/v1
```

### त्वरित उदाहरण

```bash
curl -X POST https://api.tronrental.com/v1/energy/buy \
  -H "X-API-Key: your_api_key" \
  -H "Content-Type: application/json" \
  -d '{
    "address": "TYourTronAddress...",
    "energy_amount": 65000,
    "duration_hours": 1
  }'
```

```json Response
{
  "order_id": 12345,
  "address": "TYourTronAddress...",
  "energy_amount": 65000,
  "duration_hours": 1,
  "price_trx": "2.75",
  "status": "pending"
}
```

## अगले कदम

<Steps>
  <Step title="अपनी API कुंजी प्राप्त करें">
    [tronrental.com](https://tronrental.com/auth/register) पर साइन अप करें और [डैशबोर्ड](https://tronrental.com/dashboard/api) में API कुंजी बनाएं।
  </Step>
  <Step title="वर्तमान कीमतें देखें">
    वर्तमान एनर्जी और बैंडविड्थ कीमतें देखने के लिए `GET /api/v1/prices` कॉल करें।
  </Step>
  <Step title="अपनी पहली खरीदारी करें">
    TRON एड्रेस के लिए एनर्जी खरीदने हेतु `POST /api/v1/energy/buy` का उपयोग करें।
  </Step>
</Steps>
