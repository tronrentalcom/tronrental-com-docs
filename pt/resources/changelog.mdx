## Changelog

### Março 2026

- **Pacotes de Transferência** — pacotes pré-pagos de transferências USDT
- **BW Guarantee** — delegação opcional de largura de banda para Smart Mode e Pacotes de Transferência
- **Webhooks** — notificações em tempo real para eventos da conta
- **API de Saque** — saque TRX para qualquer endereço TRON

### Fevereiro 2026

- **Smart Mode** — energia automática para transferências USDT
- **API de Faturas** — crie faturas de pagamento para energia
- **Chaves de API** — suporte a lista de IPs permitidos
- **API de Largura de Banda** — compre largura de banda para endereços TRON

### Janeiro 2026 — Lançamento Inicial

- API de compra de energia
- Gerenciamento de conta (saldo, depósitos, perfil)
- Preços em tempo real
