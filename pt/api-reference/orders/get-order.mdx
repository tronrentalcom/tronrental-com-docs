## Obter pedido

Retorna o status atual e os detalhes completos de um pedido de energia ou largura de banda.

### Parâmetros de caminho

<ParamField path="order_id" type="number" required>
  ID do pedido — o `id` retornado por Buy Energy ou Buy Bandwidth.
</ParamField>

### Resposta

Retorna o objeto completo do pedido, incluindo `status`, `type`, `volume`, `price_trx`, `delegate_txid` e carimbos de data/hora.

### Status do pedido

| Status | Descrição |
|--------|-------------|
| `pending` | Pedido recebido, delegação em andamento |
| `filled` | Energia ou largura de banda delegada com sucesso |
| `failed` | A delegação falhou |
| `reclaimed` | Largura de banda devolvida após o período de aluguel (apenas pedidos de largura de banda) |

### Exemplo

```bash
curl https://api.tronrental.com/v1/orders/12345 \
  -H "X-API-Key: your_api_key"
```
