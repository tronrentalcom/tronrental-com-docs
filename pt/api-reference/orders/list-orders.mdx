## Listar pedidos

Retorna uma lista paginada dos seus pedidos, os mais recentes primeiro.

### Parâmetros de consulta

<ParamField query="page" type="number">
  Número da página (padrão 1)
</ParamField>

<ParamField query="page_size" type="number">
  Resultados por página (padrão 50, máximo 100)
</ParamField>

<ParamField query="status" type="string">
  Filtrar por status (ex.: `filled`, `pending`, `failed`)
</ParamField>

<ParamField query="type" type="string">
  Filtrar por tipo de pedido — `energy` ou `bandwidth`
</ParamField>

### Resposta

Retorna `orders` (array de objetos de pedido), `total`, `page` e `page_size`.

### Exemplo

```bash
curl "https://api.tronrental.com/v1/orders?type=energy&status=filled&page=1" \
  -H "X-API-Key: your_api_key"
```
