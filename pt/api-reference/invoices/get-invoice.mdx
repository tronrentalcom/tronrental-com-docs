## Consultar Fatura

Retorna detalhes e status atual de uma fatura.

### Parâmetros de caminho

<ParamField path="invoice_id" type="number" required>
  ID da fatura
</ParamField>

### Resposta

Retorna o objeto completo da fatura incluindo `status`, `payment_address`, `price_trx`, `price_usdt`, `tx_hash` e timestamps.

### Status da fatura

| Status | Descrição |
|--------|-----------|
| `pending` | Aguardando pagamento |
| `paid` | Pagamento recebido, delegação em andamento |
| `delegated` | Energia delegada com sucesso |
| `expired` | Nenhum pagamento recebido antes da expiração |
| `failed` | Delegação falhou (pagamento reembolsado) |

### Exemplo

```bash
curl https://api.tronrental.com/v1/invoices/12345 \
  -H "X-API-Key: your_api_key"
```
