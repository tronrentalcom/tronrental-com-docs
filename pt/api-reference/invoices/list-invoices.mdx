## Listar Faturas

Retorna uma lista paginada das suas faturas.

### Parâmetros de consulta

<ParamField query="page" type="number" default="1">
  Número da página
</ParamField>

<ParamField query="status" type="string">
  Filtrar por status: `pending`, `paid`, `delegated`, `expired`, `failed`
</ParamField>

### Resposta

Retorna um objeto paginado com array `invoices` e contagem `total`.

### Exemplo

```bash
curl "https://api.tronrental.com/v1/invoices/my?page=1&status=delegated" \
  -H "X-API-Key: your_api_key"
```
