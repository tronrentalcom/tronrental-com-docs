## Consultar Preços

Retorna os preços de mercado atuais para energia e largura de banda.

**Este endpoint é público — nenhuma autenticação necessária.**

### Resposta

<ResponseField name="energy_price_sun" type="number">
  Preço atual da energia em SUN por unidade
</ResponseField>

<ResponseField name="energy_price_1k_trx" type="string">
  Preço para 1.000 unidades de energia em TRX
</ResponseField>

<ResponseField name="bw_price_1k_trx" type="string">
  Preço para 1.000 unidades de largura de banda em TRX
</ResponseField>

<ResponseField name="trx_usd_rate" type="string">
  Taxa de câmbio atual TRX/USD
</ResponseField>

<ResponseField name="burn_sun_per_energy" type="number">
  Taxa de queima da rede — SUN por unidade de energia (atualmente 100)
</ResponseField>

<ResponseField name="min_energy" type="number">
  Quantidade mínima de energia por pedido
</ResponseField>

<ResponseField name="max_energy" type="number">
  Quantidade máxima de energia por pedido
</ResponseField>

### Exemplo

<CodeGroup>
```bash cURL
curl https://api.tronrental.com/v1/prices
```

```python Python
import requests
resp = requests.get("https://api.tronrental.com/v1/prices")
print(resp.json())
```

```javascript JavaScript
const resp = await fetch("https://api.tronrental.com/v1/prices");
const data = await resp.json();
console.log(data);
```
</CodeGroup>
