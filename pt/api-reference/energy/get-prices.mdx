## Consultar Precos

Retorna os precos de mercado atuais para energia e largura de banda.

**Este endpoint e publico — nenhuma autenticacao necessaria.**

### Resposta

<ResponseField name="energy_trx" type="object">
  Preco de energia em TRX para 65.000 unidades, por duracao (`1h`)
</ResponseField>

<ResponseField name="energy_sun" type="object">
  Preco de energia em SUN por 1 unidade, por duracao (`1h`)
</ResponseField>

<ResponseField name="energy_usd" type="object">
  Preco de energia em USD para 65.000 unidades, por duracao (`1h`)
</ResponseField>

<ResponseField name="energy_volume" type="number">
  Volume utilizado para calculo do preco de energia (65.000)
</ResponseField>

<ResponseField name="bandwidth_trx" type="object">
  Preco de largura de banda em TRX para 350 unidades, por duracao (`1h`)
</ResponseField>

<ResponseField name="bandwidth_sun" type="object">
  Preco de largura de banda em SUN por 1 unidade, por duracao (`1h`)
</ResponseField>

<ResponseField name="bandwidth_usd" type="object">
  Preco de largura de banda em USD para 350 unidades, por duracao (`1h`)
</ResponseField>

<ResponseField name="bandwidth_volume" type="number">
  Volume utilizado para calculo do preco de largura de banda (350)
</ResponseField>

<ResponseField name="trx_usd_rate" type="string">
  Taxa de cambio atual TRX/USD
</ResponseField>

<ResponseField name="burn_cost_trx" type="string">
  Custo de queimar TRX para 65.000 de energia (sem aluguel)
</ResponseField>

<ResponseField name="burn_cost_usd" type="string">
  Mesmo custo de queima em USD
</ResponseField>

<ResponseField name="savings_percent" type="string">
  Porcentagem de economia ao alugar vs queimar
</ResponseField>

### Exemplo

<CodeGroup>
```bash cURL
curl https://api.tronrental.com/v1/prices
```

```python Python
import requests

print(resp.json())
```

```javascript JavaScript

```
</CodeGroup>
