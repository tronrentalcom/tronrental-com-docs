## Pacotes de Transferência

Os Pacotes de Transferência são pacotes pré-pagos de transferências USDT. Compre um pacote de N transferências, e cada transferência USDT de saída do endereço recebe energia automaticamente até que o pacote seja esgotado.

### Como funciona

1. **Compre um pacote** para um endereço TRON específico (ex.: 500 transferências)
2. Cada transferência USDT de saída **recebe energia automaticamente**
3. O contador do pacote diminui a cada transferência
4. Quando o pacote é esgotado, o monitoramento é interrompido

### Preços

| Tamanho do pacote | Preço por transferência |
|-------------------|------------------------|
| 1 – 10.000 transferências | 2,7 TRX |
| BW Guarantee (opcional) | +0,3 TRX/transferência |

**Presets de seleção rápida:** 500, 1.000, 5.000, 10.000

### Pacotes de Transferência vs Smart Mode

| Recurso | Pacotes de Transferência | Smart Mode |
|---------|--------------------------|------------|
| Cobrança | Pré-pago (único) | Assinatura diária + por transferência |
| Duração | Até esgotar as transferências | Contínuo |
| Ideal para | Volume previsível | Uso contínuo |

<Warning>
  Um endereço não pode ter um Pacote de Transferência ativo e o Smart Mode ao mesmo tempo.
</Warning>
