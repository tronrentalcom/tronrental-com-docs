## Criar Saque

<div>
  `POST /api/v1/account/withdrawal`
</div>

Saca TRX para qualquer endereço TRON. Taxa: 1 TRX.

### Corpo da requisição

<ParamField body="to_address" type="string" required>
  Endereço TRON de destino
</ParamField>

<ParamField body="amount_trx" type="string" required>
  Valor a sacar em TRX
</ParamField>

<ParamField body="totp_code" type="string">
  Código 2FA (se o 2FA estiver ativado)
</ParamField>

### Exemplo

```bash
curl -X POST https://api.tronrental.com/v1/account/withdrawal \
  -H "X-API-Key: your_api_key" \
  -H "Content-Type: application/json" \
  -d '{
    "to_address": "TDestinationAddress...",
    "amount_trx": "100"
  }'
```

```json Response
{
  "id": 56,
  "to_address": "TDestinationAddress...",
  "amount_trx": "100.00",
  "fee_trx": "1.00",
  "status": "completed",
  "tx_hash": "a1b2c3d4e5f6...",
  "created_at": "2026-03-05T12:00:00Z"
}
```

## Listar Saques

<div>
  `GET /api/v1/account/withdrawals`
</div>

Retorna seu histórico de saques.

```bash
curl https://api.tronrental.com/v1/account/withdrawals \
  -H "X-API-Key: your_api_key"
```

```json Response
[
  {
    "id": 56,
    "to_address": "TDestinationAddress...",
    "amount_trx": "100.00",
    "fee_trx": "1.00",
    "status": "completed",
    "tx_hash": "a1b2c3d4e5f6...",
    "created_at": "2026-03-05T12:00:00Z"
  }
]
```

### Status dos saques

| Status | Descrição |
|--------|-----------|
| `pending` | Na fila para processamento |
| `completed` | TRX enviado, tx_hash disponível |
| `failed` | Falhou, valor reembolsado |
