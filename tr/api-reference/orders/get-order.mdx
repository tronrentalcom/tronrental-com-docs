## Sipariş Durumu

Bir enerji veya bant genişliği siparişinin güncel durumunu ve tüm ayrıntılarını döndürür.

### Yol parametreleri

<ParamField path="order_id" type="number" required>
  Sipariş ID'si — Buy Energy veya Buy Bandwidth tarafından döndürülen `id`.
</ParamField>

### Yanıt

`status`, `type`, `volume`, `price_trx`, `delegate_txid` ve zaman damgaları dahil tüm sipariş nesnesini döndürür.

### Sipariş durumları

| Durum | Açıklama |
|--------|-------------|
| `pending` | Sipariş alındı, delegasyon devam ediyor |
| `filled` | Enerji veya bant genişliği başarıyla delege edildi |
| `failed` | Delegasyon başarısız oldu |
| `reclaimed` | Kiralama süresi sonunda bant genişliği geri alındı (yalnızca bant genişliği siparişleri) |

### Örnek

```bash
curl https://api.tronrental.com/v1/orders/12345 \
  -H "X-API-Key: your_api_key"
```
