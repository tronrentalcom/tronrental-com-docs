## Enerji Satin Al

Belirtilen TRON adresine enerji devredilir. Maliyet hesap bakiyenizden dusulur.

### Istek govdesi

<ParamField body="address" type="string" required>
  Hedef TRON adresi (T... formati, 34 karakter)
</ParamField>

<ParamField body="energy_amount" type="number" required>
  Devredilecek enerji miktari (min 32.000, maks 5.000.000)
</ParamField>

<ParamField body="duration_hours" type="number" required>
  Kiralama suresi saat cinsinden (1 veya 24)
</ParamField>

### Yanit

<ResponseField name="order_id" type="number">
  Benzersiz siparis tanimlayicisi
</ResponseField>

<ResponseField name="address" type="string">
  Hedef adres
</ResponseField>

<ResponseField name="energy_amount" type="number">
  Devredilen enerji
</ResponseField>

<ResponseField name="duration_hours" type="number">
  Kiralama suresi
</ResponseField>

<ResponseField name="price_trx" type="string">
  TRX cinsinden toplam maliyet
</ResponseField>

<ResponseField name="status" type="string">
  Siparis durumu: `pending`, `filled`, `failed`
</ResponseField>

### Ornek

<CodeGroup>
```bash cURL
curl -X POST https://api.tronrental.com/v1/energy/buy \
  -H "X-API-Key: your_api_key" \
  -H "Content-Type: application/json" \
  -d '{
    "address": "TYourRecipientAddress1234567890abc",
    "energy_amount": 65000,
    "duration_hours": 1
  }'
```

```python Python
import requests

resp = requests.post(
    "https://api.tronrental.com/v1/energy/buy",
    headers={"X-API-Key": "your_api_key"},
    json={
        "address": "TYourRecipientAddress1234567890abc",
        "energy_amount": 65000,
        "duration_hours": 1,
    },
)
print(resp.json())
```

```javascript JavaScript
const resp = await fetch("https://api.tronrental.com/v1/energy/buy", {
  method: "POST",
  headers: {
    "X-API-Key": "your_api_key",
    "Content-Type": "application/json",
  },
  body: JSON.stringify({
    address: "TYourRecipientAddress1234567890abc",
    energy_amount: 65000,
    duration_hours: 1,
  }),
});
console.log(await resp.json());
```

<Tip>
  **65.000 enerji**, halihazirda USDT bulunan bir adrese tek bir USDT transferini karsilar.
  Ilk kez USDT alacak alicilar icin **131.000** kullanin (yeni depolama alani olusturma).
</Tip>
