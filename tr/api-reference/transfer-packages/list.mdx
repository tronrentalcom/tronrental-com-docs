## Paketlerimi Listele

Tum transfer paketlerinizi dondurur.

### Yanit

`id`, `address`, `total_transfers`, `remaining_transfers`, `status` ve `created_at` iceren paket nesneleri dizisi.

### Paket durumlari

| Durum | Aciklama |
|--------|-------------|
| `pending_payment` | Fatura odemesi bekleniyor |
| `active` | Adres izleniyor, transferler mevcut |
| `completed` | Tum transferler kullanildi |
| `cancelled` | Kullanici tarafindan iptal edildi |

### Ornek

```bash
curl https://api.tronrental.com/v1/transfer-packages/my \
  -H "X-API-Key: your_api_key"
```
