## Faturalari Listele

Sayfalandirilmis fatura listenizi dondurur.

### Sorgu parametreleri

<ParamField query="page" type="number" default="1">
  Sayfa numarasi
</ParamField>

<ParamField query="status" type="string">
  Duruma gore filtrele: `pending`, `paid`, `delegated`, `expired`, `failed`
</ParamField>

### Yanit

`invoices` dizisi ve `total` sayisini iceren sayfalandirilmis bir nesne dondurur.

### Ornek

```bash
curl "https://api.tronrental.com/v1/invoices/my?page=1&status=delegated" \
  -H "X-API-Key: your_api_key"
```
