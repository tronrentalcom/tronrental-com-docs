## Otomatik Yenilenen Kiralamaları Listele

Otomatik yenilenen kiralamalarınızı en yeniden başlayarak döndürür.

### Sorgu parametreleri

<ParamField query="status" type="string">
  Duruma göre filtrele: `active`, `cancelled` veya `ended_insufficient_balance`
</ParamField>

### Yanıt

`rentals` döndürür — `status`, `volume`, `daily_trx`, `total_charged_trx`, `next_billing_at` ve zaman damgaları içeren kiralama nesneleri dizisi.

### Kiralama durumları

| Durum | Açıklama |
|--------|-------------|
| `active` | Delege edildi ve günlük ücretlendiriliyor |
| `cancelled` | Kullanıcı (veya admin) tarafından iptal edildi |
| `ended_insufficient_balance` | Otomatik sona erdi — bakiye ertesi günü karşılamadı |

### Örnek

```bash
curl "https://api.tronrental.com/v1/bandwidth/rentals?status=active" \
  -H "X-API-Key: your_api_key"
```
