## 1. API anahtarınızı alın

Kaydolun ve [tronrental.com/dashboard/api](https://tronrental.com/dashboard/api) adresinde bir API anahtarı oluşturun.

## 2. Fiyatları kontrol edin

```bash
curl https://api.tronrental.com/v1/prices
```

```json
{
  "energy_trx": { "1h": "1.79" },
  "energy_sun": { "1h": "27.54" },
  "energy_volume": 65000,
  "energy_fixed_fee_trx": "0.2",
  "bandwidth_trx": { "1h": "0.34", "1d": "0.42" },
  "bandwidth_sun": { "1h": "400", "1d": "630" },
  "bandwidth_volume": 350,
  "trx_usd_rate": "0.234",
  "burn_cost_trx": "6.5",
  "savings_percent": "72.5"
}
```

Enerji kiralaması **yalnızca 1 saat** (`"1h"`). Bandwidth için `"1h"` veya `"1d"`. `energy_trx` yalnızca kiralama fiyatıdır; ödeme sırasında `energy_fixed_fee_trx` (0.2 TRX) eklenir.

## 3. TRX yatırın

Yatırma adresinizi alın:

```bash
curl https://api.tronrental.com/v1/account/deposit \
  -H "X-API-Key: your_api_key"
```

Dönen `deposit_address` adresine TRX veya USDT gönderin. Yatırımlar otomatik olarak hesaba geçer.

## 4. Enerji satın alın

```bash
curl -X POST https://api.tronrental.com/v1/energy/buy \
  -H "X-API-Key: your_api_key" \
  -H "Content-Type: application/json" \
  -d '{
    "target_address": "TYourRecipientAddress1234567890abc",
    "volume": 65000,
    "duration": "1h"
  }'
```

```json
{
  "id": 12345,
  "price_trx": "2.75",
  "status": "pending",
  "txid": null,
  "reclaim_at": null,
  "activation_cost_trx": "0"
}
```

Enerji birkaç saniye içinde hedef adrese devredilir.

## 5. Doğrulayın

Sipariş durumunu kontrol edin:

```bash
curl https://api.tronrental.com/v1/orders \
  -H "X-API-Key: your_api_key"
```

```json
{
  "orders": [
    {
      "id": 12345,
      "type": "energy",
      "target_address": "TYourRecipientAddress1234567890abc",
      "volume": 65000,
      "duration": "1h",
      "price_trx": "2.75",
      "status": "filled",
      "delegate_txid": "abc123...",
      "reclaim_at": null,
      "source": "api",
      "created_at": "2026-06-09T12:00:00Z"
    }
  ],
  "total": 1,
  "page": 1,
  "page_size": 50
}
```

<Tip>
  **65.000 enerji**, zaten USDT tutan bir adrese tek bir USDT transferi için yeterlidir.
  İlk kez USDT alan adresler için **131.000 enerji** kullanın.
</Tip>

## Kod örnekleri

<CodeGroup>
```python Python
import requests

API_KEY = "your_api_key"
BASE = "https://api.tronrental.com/v1"
headers = {"X-API-Key": API_KEY}

resp = requests.post(
    f"{BASE}/energy/buy",
    headers=headers,
    json={
        "target_address": "TYourRecipientAddress1234567890abc",
        "volume": 65000,
        "duration": "1h",
    },
)
print(resp.json())
```

```javascript JavaScript
const API_KEY = "your_api_key";
const BASE = "https://api.tronrental.com/v1";

const res = await fetch(`${BASE}/energy/buy`, {
  method: "POST",
  headers: {
    "X-API-Key": API_KEY,
    "Content-Type": "application/json",
  },
  body: JSON.stringify({
    target_address: "TYourRecipientAddress1234567890abc",
    volume: 65000,
    duration: "1h",
  }),
});
console.log(await res.json());
```
