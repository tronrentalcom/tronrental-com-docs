## Get Balance

Returns your current TRX balance.

### Response

<ResponseField name="balance_trx" type="string">
  Account balance in TRX (decimal string)
</ResponseField>

### Example

```bash
curl https://api.tronrental.com/v1/account/balance \
  -H "X-API-Key: your_api_key"
```

```json
{
  "balance_trx": "156.50"
}
```
