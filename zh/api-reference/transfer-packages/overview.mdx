## 转账套餐

转账套餐是预付费的 USDT 转账包。购买 N 笔转账的套餐后，该地址每笔 USDT 转出都会自动获得能量，直到套餐用完。

### 工作原理

1. **购买套餐**，绑定特定 TRON 地址（例如 500 笔转账）
2. 每笔 USDT 转出**自动获得能量**
3. 套餐计数器每笔转账递减
4. 套餐用尽后，监控停止

### 定价

| 套餐规模 | 每笔转账价格 |
|-------------|-------------------|
| 1 – 10,000 笔转账 | 2.7 TRX |
| 带宽保障（可选） | +0.3 TRX/笔 |

**快捷预设：** 500、1,000、5,000、10,000

### 转账套餐 vs 智能模式

| 功能 | 转账套餐 | 智能模式 |
|---------|------------------|------------|
| 计费方式 | 预付费（一次性） | 每日订阅费 + 每笔转账费 |
| 时长 | 直至转账用完 | 持续有效 |
| 适用场景 | 可预测的交易量 | 持续使用 |

<Warning>
  同一地址不能同时拥有活跃的转账套餐和智能模式。
</Warning>
