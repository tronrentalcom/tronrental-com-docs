## 智能模式

智能模式自动为被监控地址的每笔 USDT 转出提供能量。无需手动调用 API — 只需激活即可开始转账。

### 工作原理

1. **激活**某个 TRON 地址的智能模式
2. 每当该地址发送 USDT 时，能量将**自动委托**
3. 每日订阅费 + 每笔转账费从您的余额中扣除
4. 余额不足时，订阅将暂停（不会被删除）

### 定价

| 费用 | 金额 |
|-----|--------|
| 每日订阅费 | 3.0 TRX/天 |
| 每笔 USDT 转账 | 2.7 TRX × 倍数 |
| 带宽保障（可选） | +0.3 TRX/笔 |

转账倍数：已持有 USDT 的地址（65K 能量）为 `1`，首次接收的地址（131K 能量）为 `2`。

<Tip>
  **带宽保障**（默认启用）会委托 350 带宽来覆盖交易的带宽费用。
  如不启用，发送方需要约 0.35 TRX 的带宽或冻结的 TRX。
</Tip>

<Info>
  以上为默认价格。请使用 [`GET /smart-mode/fees`](/api-reference/smart-mode/get-fees) 查看您的实际费率 — 可能适用自定义定价。
</Info>

### 限制

- 每个账户最多 5 个活跃订阅
- 大流量用户可获取自定义定价（请联系客服）
