## 购买带宽

将带宽委托给指定的 TRON 地址。

### 请求体

<ParamField body="address" type="string" required>
  目标 TRON 地址
</ParamField>

<ParamField body="bandwidth_amount" type="number" required>
  带宽数量（最小 350，最大 100,000）
</ParamField>

<ParamField body="duration_hours" type="number" required>
  租赁时长，以小时计
</ParamField>

### 响应

返回包含 `order_id`、`status` 和 `price_trx` 的订单对象。

### 定价

带宽价格：`(volume / 1000) × 9.6 / 24 + 0.2 TRX`

所有带宽订单均收取 0.2 TRX 的固定费用。

### 示例

```bash
curl -X POST https://api.tronrental.com/v1/bandwidth/buy \
  -H "X-API-Key: your_api_key" \
  -H "Content-Type: application/json" \
  -d '{
    "address": "TTargetAddress...",
    "bandwidth_amount": 1000,
    "duration_hours": 24
  }'
```

```json Response
{
  "order_id": 5678,
  "address": "TTargetAddress...",
  "bandwidth_amount": 1000,
  "duration_hours": 24,
  "price_trx": "9.80",
  "status": "pending"
}
```
